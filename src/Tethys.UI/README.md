# Getting started

1. Install node 8 or higher
2. Install npm 5 or higher
3. clone the repository `git clone https://github.com/devCrossNet/vue-starter`
4. change directory `cd vue-starter`
5. Install dependencies: `npm install`
6. Run `npm run dev`
